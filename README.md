# Data_Storytelling_Using_StreamLit
• Built an interactive data storytelling dashboard using Streamlit and Plotly to analyse global
workforce reductions across industries and countries.
• Performed data cleaning, preprocessing, KPI generation, trend analysis, and visualization on realworld layoff datasets.
• Generated insights into industry impact, temporal trends, and funding-layoff relationships

# Kaggle Dataset link:
